Online in 5 minuten?

  • Stap 1: Maak een account aan op GitHub.com. Zorg ervoor dat je email-adres geverifieerd is door op de link in de mail te klikken die je van GitHub zal ontvangen.
  • Stap 2: Installeer GitHub Desktop
  • Stap 3: Maak een repository aan via de startpagina GitHub website.
  • Stap 4: Je opent op de GitHub website de pagina van je repository en klikt bovenaan links op Set up in desktop
  • Stap 5: GitHub desktop opent, en je selecteert de folder waarin GitHub desktop de projectfolder met dezelfde naam als de repositorynaam zal aanmaken. Let op: je selecteert hier dus de bovenliggende folder voor jouw nieuwe projectfolder, NIET de projectfolder zelf.
  • Stap 6: Je opent de projectfolder in kwestie en kopieert er de online te plaatsen bestanden naartoe, of je begint in deze folder te werken aan een nieuw project. Vanaf nu zal je in deze folder je project afwerken.
  • Stap 7: Nadat je hiermee klaar bent keer je terug naar GitHub Desktop. Je selecteert jouw repository. Links onderaan voeg je een bondige beschrijvende titel toe voor je wijziging en klik je op Commit to master. Daarna klik je op de knop Sync rechts bovenaan.
  • Stap 8: Op GitHub.com open je de settings van je repository. Je scrollt naar beneden tot aan de sectie "GitHub pages". Je selecteert er master branch Use the master branch for GitHub Pages en klikt op "Save". Een minuutje geduld en je site staat online, via de link die in de settings wordt getoond.
  • Telkens er een nieuwe versie van je site klaar is herhaal je stap 7.
